# LPDIM_PHP_Tempura

## Pré-requis 
* Symfony 5.2
* Composer 

## Installation 
``composer update``

### Yarn 

Si vous possédez déjà Yarn vous pouvez directement faire : 
``yarn install``

Sinon : 
``npm install --global yarn``
``yarn install``

#### Pour actualiser les scripts et feuilles de styles modifiées depuis /assets :
``yarn encore dev``
